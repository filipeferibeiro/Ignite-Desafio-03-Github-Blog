# Ignite-Desafio-03-Github-Blog

Projeto Desafio 03 do Ignite

## Sobre o projeto

O projeto consiste em um Blog, em que o usuário pode acessar as Issues deste repositório do GitHub, com a possibilidade de realizar pesquisas.

## Tecnologias

1. ReactJS
2. TypeScript
3. Vite
4. Axios
